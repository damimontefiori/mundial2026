# Numeración oficial — Álbum Panini FIFA World Cup 2026™

> Verificado contra el **checklist oficial del set base** (Panini, abril 2026) y los **materiales oficiales de la promoción Coca-Cola × Panini**. Esta es la corrección de las tres versiones que te dieron las IAs.

## Totales verificados

- **Set base del álbum: 980 figuritas**, repartidas así:
  - **1** logo Panini → `00`
  - **19** especiales `FWC` → `FWC1` a `FWC19`
  - **960** de selecciones → 48 equipos × 20 figuritas
- **Promoción Coca-Cola (set aparte): 12 figuritas** → `CC1` a `CC12`
  - **No vienen en los sobres normales.** Se consiguen bajo la etiqueta de botellas Coca-Cola / Coca-Cola Zero 20 oz o en la app de Panini. El álbum trae una **página reservada** para pegarlas.

> Para "completar" físicamente el álbum hacen falta las 980 del set base **más** las 12 de Coca-Cola = **992 espacios**.

## Correcciones respecto a las tres IAs

- El total **no es 994**: el set base es **980**. (Esa versión sumaba mal los especiales).
- El set Coca-Cola es **`CC1`–`CC12` (12 figuritas)**, **no `CC1`–`CC14`**. El "C1–C14" corresponde al álbum **Qatar 2022**, no a 2026.
- `00` + `FWC1`–`FWC19` (20 piezas) ✅ está bien en las tres.
- Las 48 selecciones y sus códigos ✅ están bien.

## Notas útiles

- **Letra "O" vs cero:** en `SCO` (Escocia), `COL` (Colombia), `CRO` (Croacia) y `COD` (RD del Congo) el carácter es la **letra O**, no un cero.
- **Códigos que pueden confundir:** `RSA` = Sudáfrica · `KOR` = Corea del Sur · `CIV` = Costa de Marfil · `CUW` = Curazao · `CPV` = Cabo Verde · `COD` = RD del Congo · `KSA` = Arabia Saudita · `QAT` = Catar.
- **Orden de grupos:** sigue la disposición del álbum (equipos agrupados de a 4 según el sorteo del Mundial). Si querés ordenarlo alfabético por código, avisame.
